# gmod-lobotomy-death-sound-mod
makes death sound lobotomy sound


put folder in C:\Program Files (x86)\Steam\steamapps\common\GarrysMod\garrysmod\addons (addons folder)

sometimes its really quiet because one of the death sounds is the beep sound, and it's volume is controlled by console cvar "suitvolume" which is defaulted to 0.25
